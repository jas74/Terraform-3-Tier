# Terraform 3 Tier Application

# Services and Tools Used In this 3 Tier Application Project.
```
DevOps Tools Used :-
  - Git
    - For Source code management and for version control.
    - Basic Commands used :-
        - Git clone
        - Git remote add
        - Git add .
        - Git commit
        - Git push
        - Git pull

  - Terraform
    - For Provisioning Cloud Infrastructure Using :-
        - Terraform Functions
        - Terraform Provisioners
        - Terraform Workspace
        - Terraform Module
        - Terraform Backend
        - Terraform TFVARs

```
```

AWS Service Provisioned :-
  - S3
    - For Terraform remote backend
  - EC2
    - For creating :-
      - Virtual Machine
      - Load Balancer
      - Target Group
      - Launch Template
      - Auto Scaling
      - User data
  - VPC
    - For creating :-
      - Private and Public Subnets
      - Internet and NAT gateway
      - Elastic IP
      - Route Table
  - Secret Manager
    - For storing RDS :-
      - Database name
      - Database username
      - Database password
  - RDS
    - For creating database instance

```
```

Linux Services Used :-
  - Web Services :-
    - NGINX
    - Apache Tomcat 
  - Database Service :-
    - mysql

```

```

Shell Scripting Commands :-
  - sed
  - awk
  - xargs
```
